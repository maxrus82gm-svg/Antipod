# КАРТА ЧТЕНИЯ

Перед началом любой задачи читать:

```text
Старт.md
Documentation/Агентский_цикл.md
```

Существующие связанные документы:

- [[Documentation/История_задач]]
- [[Documentation/Список_правил]]

Тематическое чтение:

```text
UI / интерфейс -> Documentation/Rules/UI_RULES.md + Documentation/Rules/ENCODING_RULES.md
JSON / конфиги -> Documentation/Rules/JSON_CONTRACT.md + Documentation/Rules/ENCODING_RULES.md
Сетевая синхронизация -> Documentation/Rules/NETWORK_SYNC_RULES.md
Git / commit / review -> Documentation/Rules/GIT_REVIEW_RULES.md
Кодировка / кириллица -> Documentation/Rules/ENCODING_RULES.md
История / статус / прошлые решения -> Documentation/История_задач.md
```

Если нужного документа нет, указать это в отчете.
